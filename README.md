# Task 2 - Stock Portfolio Tracker

## Objective

Build a Python application that calculates the total investment value of a stock portfolio.

## Features

- User inputs stock names.
- User inputs quantity.
- Calculates total investment.
- Saves report in TXT.
- Saves report in CSV.

## Concepts Used

- Dictionary
- Loops
- File Handling
- CSV Module

## Sample Output

```text
========== STOCK PORTFOLIO TRACKER ==========

Available Stocks:

AAPL : $180
TSLA : $250
GOOGL : $150
MSFT : $320
AMZN : $140

Total Investment = $1310
```

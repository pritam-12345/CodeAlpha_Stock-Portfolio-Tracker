# 📈 Stock Portfolio Tracker

A simple **Python-based Stock Portfolio Tracker** that calculates the total value of a user's stock investments using predefined stock prices. Users can enter stock symbols and quantities, and the application computes the overall portfolio value with an option to save the results to a `.txt` or `.csv` file.

This project demonstrates the use of Python dictionaries, user input handling, arithmetic operations, and basic file handling, making it an excellent beginner-friendly finance application.

---

## ✨ Features

- 📊 Track investments using predefined stock prices
- ⌨️ User-friendly console-based input
- 💰 Calculates total portfolio value instantly
- 📁 Option to save portfolio details to a `.txt` or `.csv` file
- ✅ Input validation for stock symbols and quantities
- 🧹 Clean, modular, and easy-to-understand Python code

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed

### Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Stock-Portfolio-Tracker.git
```

2. Navigate to the project folder

```bash
cd Stock-Portfolio-Tracker
```

3. Run the application

```bash
python stock_tracker.py
```

---

## 📋 How It Works

1. Enter the stock symbol (e.g., `AAPL`, `TSLA`).
2. Enter the quantity of shares owned.
3. The program retrieves the stock price from a predefined dictionary.
4. It calculates the total investment value.
5. Optionally, save the portfolio summary as a `.txt` or `.csv` file.
